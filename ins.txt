# Инструкция по проекту: Сервисный Центр SC

## Общая структура проекта

```
сервисный-центр-sc/
├── index.html              # Главная страница
├── css/
│   ├── style.css           # Основные стили
│   └── dark-theme.css      # Стили темной темы
├── js/
│   ├── loader.js           # Загрузчик блоков
│   └── floating-buttons.js # Скрипт плавающих кнопок
└── blocks/
    ├── header.html         # Шапка сайта
    ├── left-sidebar.html   # Левая боковая панель
    ├── hero.html           # Главный баннер
    ├── services.html       # Блок услуг
    ├── contact.html        # Блок контактов
    ├── right-sidebar.html  # Правая боковая панель
    └── footer.html         # Подвал сайта
```

## Описание файлов

### index.html
**Назначение**: Главный файл, который объединяет все блоки сайта.
**Что можно изменить**:
- Заголовок страницы в `<title>`
- Подключение дополнительных CSS/JS файлов
- Порядок загрузки блоков

### CSS файлы
**style.css** - основные стили сайта:
- Общие стили для всех элементов
- Адаптивная верстка
- Стили для блоков контента

**dark-theme.css** - стили темной темы:
- Цветовая схема
- Стили для конкретных элементов в темном режиме
- Анимации и эффекты

### JS файлы
**loader.js** - загружает содержимое блоков из отдельных файлов:
- Можно добавлять/удалять блоки
- Изменять порядок загрузки

**floating-buttons.js** - управляет плавающими кнопками:
- Анимации кнопок
- Поведение при клике

### Блоки (папка blocks)

#### header.html
**Назначение**: Шапка сайта с контактами и навигацией.
**Что можно изменить**:
- Название компании
- Телефон и адрес
- Пункты меню навигации
- Ссылку на Яндекс.Карты

#### left-sidebar.html
**Назначение**: Левая боковая панель с дополнительными услугами.
**Что можно изменить**:
- Заголовки и текст блоков
- Добавить новые блоки услуг
- Изменить изображения (через data URI или ссылки)

#### hero.html
**Назначение**: Главный баннер с призывом к действию.
**Что можно изменить**:
- Основной заголовок
- Описание под заголовком
- Текст кнопки (если добавить)

#### services.html
**Назначение**: Основной блок с услугами сервисного центра.
**Что можно изменить**:
- Добавить/удалить услуги
- Изменить иконки услуг
- Отредактировать описания услуг

#### contact.html
**Назначение**: Блок с контактной информацией.
**Что можно изменить**:
- Телефон и адрес
- Часы работы
- Добавить карту (раскомментировать код)

#### right-sidebar.html
**Назначение**: Правая боковая панель с дополнительной информацией.
**Что можно изменить**:
- Блок "Новое"
- Блок "Новинки"
- Блок "Статьи"
- Добавить новые информационные блоки

#### footer.html
**Назначение**: Подвал сайта.
**Что можно изменить**:
- Текст копирайта
- Добавить ссылки на соцсети
- Добавить дополнительную информацию

## Как вносить изменения

1. **Для изменения контента**:
   - Откройте соответствующий файл в папке `blocks`
   - Отредактируйте HTML-код
   - Сохраните изменения

2. **Для изменения стилей**:
   - Откройте файлы в папке `css`
   - Измените CSS-свойства
   - Сохраните изменения

3. **Для изменения структуры**:
   - Откройте `index.html`
   - Добавьте/удалите блоки
   - Обновите `loader.js` для загрузки новых блоков

## Особенности проекта

- **Модульная структура**: Каждый блок находится в отдельном файле
- **Темная тема**: Сайт использует темный цветовой scheme
- **Адаптивность**: Сайт корректно отображается на мобильных устройствах
- **Плавающие кнопки**: Кнопка телефона и доната с анимациями
- **Карта**: Интеграция с Яндекс.Картами (закомментирована)

## Дополнительные возможности

1. **Добавление карты**:
   - Раскомментируйте код в `contact.html`
   - Настройте параметры карты

2. **Добавление новых блоков**:
   - Создайте файл в папке `blocks`
   - Добавьте его загрузку в `loader.js`
   - Добавьте соответствующий div в `index.html`

3. **Изменение цветовой схемы**:
   - Отредактируйте CSS-переменные в `dark-theme.css`
